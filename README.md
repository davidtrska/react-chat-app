# react-chat-app

Simple react chatting app for me to learn basics of react

## Tech stack
- React
- Javascript
- Websockets
- CSS 
- node.js

## Running locally

### Backend
cd server
npm install
node server.js


### Frontend
cd client
npm install
npm run dev

## Environment variables
Create a `client/.env.local` file with:
VITE_BACKEND_URL=http://localhost:8080
